# Resume Bullets

## Research-leaning versions

- Implemented the core Dreamer4 world-model pipeline in JAX on a toy visual-control domain, including MAE tokenizer pretraining, shortcut-forcing latent dynamics, BC/reward finetuning, and imagination-based RL.
- Built a transformer-based action-conditioned world model with agent/task tokens, x-prediction dynamics, TD-lambda value learning, and PMPO-style policy updates regularized to a behavior prior.
- Audited the implementation against the Dreamer4 paper, distinguishing mechanism-level alignment from missing paper-scale components, and structured the repo for reproducible experiments and ablations.

## Applied ML versions

- Developed an end-to-end JAX/Flax world-model training stack with checkpointing, visualization, latent imagination rollouts, and policy/value/reward heads.
- Implemented and debugged a multi-stage visual RL pipeline spanning reconstruction pretraining, action-conditioned video prediction, imitation/reward modeling, and imagination-based policy optimization.
- Packaged a research codebase into a portfolio-ready project with experiment documentation, reproducibility guidance, and evidence-based positioning of technical scope and limitations.
